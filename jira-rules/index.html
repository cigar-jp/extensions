アイコンの凡例

  • 🈂️: 差し込み
  • 📅: 日付の指定があるもの
  • 🔜: 次のスプリントで取り掛かるので今スプリントでリファインメントしよう

エピックのステータス

  • エピックがReadyでない状態：Readyになったら、CSが色を紫に変更する
  • リファインメント待ち：リファインメントが完了したら、Techが色を青に変更する
  • 調査が必要な状態：調査が完了したら実装チケットを作成し、Techが色を青に変更する
  • 実装できる状態 / 実装中 / 実装完了

その他

  • 緊急対応：最優先で対応中
  • QA / 内部検証で検出した既存不具合
  • 調査意見箱 / CS要望 / スプリントTODOタスクとして、リファインメント分のSTPを確保している
